Guia de Configuração

Configure o Google Apps Script em 5 passos para ativar a coleta e visualização dos resultados.

1

Crie uma planilha no Google Sheets

Acesse sheets.google.com e crie uma planilha nova. Dê o nome que quiser, ex: "Perfil Comportamental — Resultados".

2

Abra o Editor de Apps Script

Na planilha, clique em Extensões → Apps Script. Uma nova aba vai abrir com o editor de código.

3

Cole o código abaixo e troque o token

Apague tudo que está no editor e cole o código abaixo. Troque o valor de ADMIN_TOKEN por uma senha de sua escolha.

const ADMIN_TOKEN = 'TROQUE-ESTA-SENHA'; // ← defina sua senha aqui function doPost(e) { try { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName('Resultados'); if (!sheet) { sheet = ss.insertSheet('Resultados'); sheet.appendRow([ 'Data/Hora','Nome','E-mail','WhatsApp', 'DISC D','DISC I','DISC S','DISC C','DISC Primário', 'Tipo Jung','Spranger 1°','Spranger 2°','Spranger 3°', 'Liderança','Liderança Scores', 'Ansiedade %','Ansiedade Nível', 'Crença Capacidade','Crença Merecimento', 'Crença Identidade','Crença Abundância' ]); } sheet.appendRow([ new Date(data.timestamp).toLocaleString('pt-BR'), data.name, data.email, data.whatsapp, data.disc_D, data.disc_I, data.disc_S, data.disc_C, data.disc_primary, data.jung_type, data.spranger_1, data.spranger_2, data.spranger_3, data.leadership_style, data.leadership_scores, data.anxiety_pct, data.anxiety_level, data.beliefs_capacidade, data.beliefs_merecimento, data.beliefs_identidade, data.beliefs_abundancia ]); return ContentService.createTextOutput('OK'); } catch(err) { return ContentService.createTextOutput('Error: ' + err.message); } } function doGet(e) { const token = e.parameter ? e.parameter.token : ''; if (token !== ADMIN_TOKEN) { return ContentService.createTextOutput( JSON.stringify({error: 'Token inválido'}) ).setMimeType(ContentService.MimeType.JSON); } const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheet = ss.getSheetByName('Resultados'); if (!sheet || sheet.getLastRow() < 2) { return ContentService.createTextOutput( JSON.stringify({data: []}) ).setMimeType(ContentService.MimeType.JSON); } const values = sheet.getDataRange().getValues(); const headers = values[0]; const rows = values.slice(1).map(row => { const obj = {}; headers.forEach((h, i) => { obj[h] = row[i]; }); return obj; }); return ContentService.createTextOutput( JSON.stringify({data: rows}) ).setMimeType(ContentService.MimeType.JSON); }
⚠️ Salve o token que você escolheu — você vai precisar dele para entrar no painel admin.
4

Implante como Web App

Clique em Implantar → Nova implantação. Em "Tipo", selecione App da Web. Configure:

  • Executar como: Eu (sua conta Google)
  • Quem tem acesso: Qualquer pessoa

Clique em Implantar e copie a URL gerada.

5

Configure o quiz para enviar dados

Abra o arquivo docs/perfil-comportamental/index.html e localize a linha:

const WEBHOOK_URL = 'SEU_WEBHOOK_URL_AQUI';

Substitua SEU_WEBHOOK_URL_AQUI pela URL do passo 4.

Após isso, toda vez que alguém completar o quiz, os dados vão para sua planilha automaticamente.
